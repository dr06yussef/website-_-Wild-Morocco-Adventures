# Plugin policy

## Required

- **WMA Core** — included in this project.
- **Polylang (free)** — French, English and Arabic content relationships, URLs and menus.

## Recommended after staging review

- **WP Mail SMTP** or **FluentSMTP** — authenticated email delivery. Configure credentials in WordPress only.
- **LiteSpeed Cache** — use only when Cap Connect confirms LiteSpeed for the selected plan.
- **Two-Factor** — administrator two-factor authentication.
- A reputable backup plugin only if an independent off-host backup is not already available.

## Optional SEO plugin

Yoast SEO or Rank Math may be added if the client needs a richer editorial SEO workflow. The theme disables its fallback Open Graph and structured-data output when either is detected, reducing duplicate metadata.

Do not install Elementor, Divi, WPBakery, duplicate caching plugins, or unmaintained add-ons.

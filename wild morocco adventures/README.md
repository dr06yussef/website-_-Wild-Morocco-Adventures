# Wild Morocco Adventures

A bespoke PHP WordPress website for multilingual, private Morocco journeys. The project contains:

- `wp-content/themes/wild-morocco-adventures` — original frontend theme.
- `wp-content/plugins/wma-core` — tours, enquiries, roles, settings, FAQs, testimonials and starter-content setup.
- `docs` — client, translation, deployment, backup and launch instructions.
- `packages` — installable ZIP files created for WordPress/cPanel.

## GitHub Pages preview

The repository root includes `index.html`, a static one-page preview that works on GitHub Pages. It uses the same theme stylesheet, JavaScript, images and videos as the WordPress website.

1. Extract the code and both video packages into the same folder before uploading.
2. Upload the extracted files and folders to the repository root; do not upload only the ZIP files.
3. In GitHub, open **Settings > Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Choose the `main` branch and `/(root)`, then save.

The GitHub Pages version is a public visual preview. WordPress administration, database content, multilingual management and enquiry processing require PHP/WordPress hosting and are not available on GitHub Pages.

## Install on a clean WordPress site

1. Use WordPress 6.5+ and select PHP 8.2 or a newer stable PHP 8 release.
2. In WordPress, upload and activate `wma-core.zip` under Plugins.
3. Upload and activate `wild-morocco-adventures.zip` under Appearance > Themes.
4. Install Polylang from WordPress.org.
5. Open Tools > Wild Morocco setup and create the optional removable starter content.
6. Open Wild Morocco > Business settings and add verified client details.
7. Configure French, English and Arabic using `docs/TRANSLATION-HANDOFF.md`.
8. Save Settings > Permalinks once, then configure HTTPS, caching, backups and SMTP.

## Administration

- **Wild Morocco > Tours**: structured trip descriptions, galleries, itinerary, budget, FAQs and related trips.
- **Wild Morocco > Enquiries**: private quote requests with workflow status and internal notes.
- **Wild Morocco > Testimonials / FAQs**: publish only verified, client-approved content.
- **Wild Morocco > Business settings**: contact, WhatsApp, quotation recipient, social links and retention.
- **Tools > Wild Morocco setup**: create or remove the original sample journeys, pages, menus, FAQs, articles, terms and media generated by the setup tool.

The `Travel Manager` role can manage content and enquiries without receiving full administrator access.

## Privacy and email

Enquiries are not publicly queryable or exposed through REST. The form uses a nonce, honeypot, rate limiting and duplicate detection. Closed enquiries are removed after the configured retention period. CSV export is permission-protected and protects spreadsheet cells against formula injection.

WordPress `wp_mail()` is used for notifications. Configure an authenticated SMTP plugin in staging; never commit mail credentials.

## Before launch

The starter-content installer provides three original, fully structured sample journeys and supporting editorial content. It deliberately avoids invented prices, reviews, awards, licences, addresses and legal claims. Review the routes and service descriptions against the client's real offer, enter verified business details, publish only approved testimonials, and have legal documents and translations reviewed by qualified professionals.
